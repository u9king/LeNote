# Unreal5课程设计

> 课程初衷：讲解一些UE5的实用小技巧，教学的同时也为今后复习留下一些痕迹
>
> 整体思路：凑满十集内容就剪辑发出，时长尽量控制在5分钟以内，每集只介绍一个知识点，用最少的素材
>
> 主要目的：从b站学了很多很多东西，所以定义为免费公益行为也做一些反哺社区的事情，提升知名度，留下一些学习痕迹以便忘记时复习，提升一些剪辑能力，不能要求再多了，再多都需要时间积累。
>
> 目前处于:1~10阶段
>
> 每次复查：需要将自己放置于初学者的角度，尽量详细的解释方法和操作步骤，标记引擎版本，既然要做就做好，不然就不做

#### 0.开篇

```
文字稿：
大家好我是u9king一个虚幻引擎的初学者。
我会为大家带来《UE5的实用小技巧》这个系列视频的初衷是为了分享一些我所学到的UE5实用小技巧，有一些非常常用，有一些偏门但是可能会有人需要而我正好学到。
我也知道很难用一个系列的视频概括所有版本的虚幻引擎技巧，更何况还在实时更新。
但是我还是决定要做这个系列
原因一方面是
我希望帮助大家了解你所疑惑的东西，很有可能虚幻已经帮大家实现了，可能换了个名字，或者地方但是只要能找到这个功能就能省去大家很多时间。
二是如果在后续版本中大家找不到想要的功能，也能有一个地方，大家能够互相讨论，一起帮忙寻找。
这是一个共同学习的过程，也欢迎你与我一起走进虚幻的世界。
```
